# Ocllama

Ollama Discord bot.

Set env vars `DISCORD_TOKEN` and `OWNER_ID` when running the bot.
Optionally `OLLAMA_URI` and `OLLAMA_PORT` too if you're not running it on localhost with default port num.

### Features

User commands
* List installed models
* Switch between them
* Chat with selected model

Admin commands
* Delete model
* Add model
